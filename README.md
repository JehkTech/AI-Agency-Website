
  # Luminous Pixels Website

  Premium AI agency marketing site for **Luminous Pixels**, built with Vite + React and Tailwind v4.

  ## Running the code

  1. Install dependencies:
     - `npm i`
  2. Start the dev server:
     - `npm run dev`
  3. Build for production:
     - `npm run build`
